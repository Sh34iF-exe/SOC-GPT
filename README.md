### A Custom GPT for SOC Analysts

SOC-GPT is an AI-powered assistant built to support **Security Operations Center (SOC) analysts** in their day-to-day operations. It helps automate repetitive tasks, enhance threat analysis, and speed up incident response while keeping analysts in control.

- **Alert Triage Assistance**

- **Incident Analysis**
  - Analyze logs, events, and telemetry
  - Map activity to MITRE ATT&CK techniques
  - Provide root-cause analysis support

- **Threat Intelligence Enrichment** (Pipeline)
  - Explain IOCs (IPs, domains, hashes)
  - Add contextual threat intelligence
  - Assist with attacker TTP analysis

- **SOC Playbook Support**
  - Recommend response actions based on playbooks
  - Generate investigation checklists
  - Assist with containment and remediation steps

- **Analyst-Friendly Explanations**
  - Clear, concise explanations for junior analysts
  - Deep-dive technical context for senior analysts
